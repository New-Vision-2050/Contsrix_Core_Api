# Laravel Reverb Quick Start

## Installation Commands

```bash
# Step 1: Install Laravel Broadcasting with Reverb
php artisan install:broadcasting

# If composer package not found, install manually:
composer require laravel/reverb
php artisan install:broadcasting
```

## Development Setup

```bash
# Terminal 1: Start Reverb WebSocket Server
php artisan reverb:start --debug

# Terminal 2: Start Queue Worker
php artisan queue:work --verbose

# Terminal 3: Run your Laravel app
php artisan serve
```

## Environment Variables

Add to `.env` (auto-generated by install command):

```env
BROADCAST_CONNECTION=reverb

REVERB_APP_ID=123456
REVERB_APP_KEY=your-app-key
REVERB_APP_SECRET=your-secret
REVERB_HOST=127.0.0.1
REVERB_PORT=8080
REVERB_SCHEME=http
```

## Test the Setup

### 1. Create an Attachment Request

```bash
curl -X POST http://localhost/api/projects/attachment-requests \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: multipart/form-data" \
  -F "name=Test Request" \
  -F "project_id=PROJECT_UUID" \
  -F "receiver_company_id=COMPANY_UUID" \
  -F "attachments[]=@/path/to/file.pdf"
```

### 2. Check Reverb Terminal

You should see:
```
[2026-04-12 11:00:00] Connection established: inbox.{user_id}
[2026-04-12 11:00:00] Broadcasting: attachment-request.created
```

### 3. Check Queue Worker Terminal

You should see:
```
[2026-04-12 11:00:00] Processing: Modules\Project\ProjectManagement\Events\AttachmentRequestCreated
```

## React Frontend Setup

```bash
# Install dependencies
npm install laravel-echo pusher-js
```

Create `src/echo.js`:

```javascript
import Echo from 'laravel-echo';
import Pusher from 'pusher-js';

window.Pusher = Pusher;

export default new Echo({
    broadcaster: 'reverb',
    key: import.meta.env.VITE_REVERB_APP_KEY,
    wsHost: import.meta.env.VITE_REVERB_HOST,
    wsPort: import.meta.env.VITE_REVERB_PORT ?? 8080,
    forceTLS: false,
    enabledTransports: ['ws', 'wss'],
});
```

Use in component:

```javascript
import echo from './echo';
import { useEffect } from 'react';

function App() {
    const userId = 'your-user-id';

    useEffect(() => {
        echo.channel(`inbox.${userId}`)
            .listen('.attachment-request.created', (e) => {
                console.log('New request:', e);
                alert(`New request: ${e.name}`);
            });
    }, [userId]);
}
```

## Production Commands

```bash
# Start Reverb in production mode
php artisan reverb:start --host=0.0.0.0 --port=8080

# Use with Supervisor (recommended)
sudo supervisorctl start reverb
sudo supervisorctl start queue-worker
```

## Troubleshooting

### Events not broadcasting?
```bash
# Check queue is running
php artisan queue:work

# Check Reverb is running
php artisan reverb:start
```

### Port already in use?
```bash
# Kill process on port 8080
# Windows:
netstat -ano | findstr :8080
taskkill /PID <PID> /F

# Linux/Mac:
lsof -ti:8080 | xargs kill -9
```

## Event Channels

| Event | Channel | Triggered When |
|-------|---------|----------------|
| `AttachmentRequestCreated` | `inbox.{receiver_user_id}` | New request created |
| `AttachmentRequestResponded` | `inbox.{sender_user_id}` | Request approved/declined |

# Restart queue worker
php artisan queue:restart
